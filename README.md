## Name Spacer
A simple C++ program that takes a name as input and displays it with spaces between each character.

# Description
This program prompts the user to enter their name and then outputs each character separated by a space. For example, if you input "John", the output will be "J o h n".
Features

User-friendly input prompt
Adds spacing between each character of the input string
Clean and simple C++ implementation

# Requirements

C++ compiler
Standard C++ libraries

# Installation
